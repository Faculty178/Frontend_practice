CSS 적용하기

인라인 스타일시트 적용

1. 태그에 직접 스타일속성을 적용하는 방법으로, 권장하지 않습니다.

임베디드 스타일시트 적용

2. 헤드태그 내에 스타일 태그를 사용하여 적용하는 방법으로, 마찬가지로 권장하지 않습니다.
html코드가 누더기가 되어버려요
인라인보다 우선순위는 낮습니다.

외부 스타일시트 적용

3. CSS파일을 연결하여 적용하는 방법으로, 권장하는 방식입니다.
Example : 'link rel="stylesheet" href="style.css"'
임베디드 스타일시트와 적용되는 숫자에 따라 우선순위가 달라집니다.
따라서 외부 스타일시트를 이용할 경우, 위 두가지 방식은 삭제해야 합니다.
